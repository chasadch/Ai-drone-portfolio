About the Project

An autonomous drone navigation system with obstacle avoidance capabilities

Autonomous Drone Navigation

This final year university project focuses on developing an autonomous drone navigation system that can navigate from point to point while avoiding obstacles in real-time. The system combines advanced computer vision techniques with sophisticated control algorithms to create a reliable and efficient navigation solution.

Our approach integrates YOLOv12 object detection with custom CNN models for obstacle detection, enabling the drone to perceive and respond to its environment with high accuracy.

Implementation

Development journey of our autonomous drone navigation system

Research & Planning

Our implementation began with extensive research into autonomous navigation systems and obstacle avoidance algorithms. We analyzed existing solutions and identified the key components needed for our system.

  • Literature review of navigation algorithms
  • Hardware component selection
  • System architecture design

Hardware Assembly

We assembled the drone platform with carefully selected components to support our navigation system requirements. This included mounting sensors, cameras, and processing units.

  • Drone frame construction
  • Sensor integration
  • Processing unit installation

AI Model Training

We trained our object detection model using YOLOv12 architecture combined with CNN for obstacle detection on a custom dataset of navigation scenarios. The training process involved multiple iterations to achieve optimal performance.

  • Dataset preparation and annotation
  • Neural network architecture design
  • Hyperparameter optimization

Performance Analysis

We conducted comprehensive performance analysis of our trained model, evaluating metrics such as detection accuracy, processing speed, and resource utilization on the drone's hardware.

  • Precision and recall evaluation
  • Processing latency measurement
  • Power consumption analysis

Field Testing

We conducted extensive field tests in various environments to validate our system's performance in real-world conditions. This included both controlled and uncontrolled testing scenarios.

  • Indoor controlled environment testing
  • Outdoor obstacle course navigation
  • System reliability assessment

Final Integration

The final phase involved integrating all components into a cohesive system, fine-tuning parameters, and preparing comprehensive documentation for our university project submission.

  • System integration and optimization
  • Documentation preparation
  • Final presentation preparation

Technology

Advanced technologies powering our autonomous navigation system

Hardware

Pixhawk 6x flight controller integrated with Jetson Nano processing unit for powerful on-board computing capabilities.

AI Model

YOLOv8 combined with custom CNN architecture for real-time obstacle detection and classification.

Perception

Multi-sensor fusion combining camera data with LiDAR for comprehensive environmental awareness.

Communication

MAVLink protocol enabling reliable and standardized communication between drone components.

Performance

85% navigation accuracy in open spaces with real-time processing capabilities for immediate response.

Software

Custom navigation algorithms developed in Python with ROS (Robot Operating System) integration.

Project Details

Academic information about our final year project

Academic Context

This autonomous drone navigation system was developed as part of our engineering department's final year project requirements for the 2024-2025 academic year.

Project Objectives

To develop a reliable autonomous navigation system with real-time obstacle detection and avoidance capabilities for unmanned aerial vehicles.

Achievements

Successfully implemented a drone system capable of navigating between points with 85% accuracy while avoiding obstacles in various environmental conditions.

Timeline

8-month development cycle from initial research and planning through to final implementation and testing.

Recognition

Received the Engineering Excellence Award for innovative application of computer vision in autonomous navigation.
